# User Management

## Overview

Manage users who can log into the tenant web app or the superadmin app.

## Types of users

- **Platform users** — superadmin accounts stored in `platform/admin`.
- **Tenant users** — company-specific accounts (future).

## Current state

- Platform admin user is seeded with hashed password.
- Tenant users are not yet implemented.
